## Deskripsi

Animeow adalah sebuah Website untuk mendapatkan informasi tentang anime dan manga. Data didapatkan dari [Jikan Moe API](https://jikan.moe/)

Disini saya belajar tentang pemanfaatan `useMemo`, React Query untuk fetching data, serta Chakra UI.

## Tech Stack

- Next JS
- Typescript
- React Query
- Chakra UI
- Jotai
