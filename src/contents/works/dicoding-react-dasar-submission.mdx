## Deskripsi

Notes App adalah sebuah Website sederhana untuk menulis catatan. Awalnya dibuat untuk memenuhi submission yang diberikan oleh Dicoding di kelas [Belajar Membuat Aplikasi Web dengan React](https://www.dicoding.com/academies/403). Pada saat itu, saya belum mengerti bagaimana membuat fitur archivenya(jika dibuat, maka kita bisa dapat poin bonus). Dan reviewer pun mengomentari bahwa Notes App saya belum mempunyai fitur archive.

Jadi setelah mendapat komentar dari reviewer, saya mencoba untuk menambahkan fitur archive ini. Alhamdulillah jadi juga. Dan beberapa waktu belakangan ini, saya juga menambahkan Supabase untuk mengelola operasi CRUD dan Authentication nya. Oh iya, saya juga memberi Form Validation dengan [Formik](https://formik.org/) dan [Yup](https://github.com/jquense/yup)

Untuk masuk ke Notes App nya, user bisa memilih salah satu diantara 2 opsi, yakni Sign In menggunakan Github atau Google.

Di project ini, saya belajar tentang basic CRUD, Jotai untuk state management, Supabase untuk BaaS(Backend as a Service), serta Form Validation.

## Tech Stack

- React JS
- Tailwind CSS
- Jotai
- Supabase
- Formik and Yup
