## Deskripsi

Studio Ghibli Resources adalah sebuah project yang berisi tentang informasi dari Studio Ghibli, seperti film-filmnya, lokasi, dan lainnya. Data didapat dari API nya [mazipan](https://github.com/mazipan/ghibli-api). Awalnya, saya membuat versi Webnya saja, tetapi kemudian saya membuat versi mobile app nya juga, dalam rangka eksplorasi ke React Native.

Saat mencoba React Native ini, saya sempat bingung soal stylingnya, apakah emang mesti manual gitu, ternyata bisa juga pake Tailwind(NativeWind). Tapi ujung-ujungnya saya pake cara manual, karena saya pikir kalau dalam pengembangan mobile app, saya ga perlu terlalu musingin soal responsive. Ada beberapa hal yang ada di versi Webnya, tapi mesti saya korbankan saat pengembangan mobile appnya. Seperti blur loading image, untuk saat ini saya belum tau gimana caranya.

Jika sebelumnya sudah biasa pake React untuk pengembangan Web, maka seharusnya ketika mencoba React Native, tidak mengalami kesulitan yang berarti, karena pada dasarnya konsepnya sama saja, sama-sama React tapi beda peruntukan. Overall, saya senang akhirnya bisa mencoba React Native ini dan langsung implementasiin ke dalam project.

Sedangkan untuk Mobile Appnya, saya menggunakan:
