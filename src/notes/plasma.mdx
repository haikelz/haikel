Jadi, saya kepikiran buat menginstall KDE Plasma setelah beralih dari XFCE. Tapi karena saya hanya membutuhkan hal-hal pentingnya saja, saya pun menginstall minimal KDE Plasma.

```bash
sudo pacman -S plasma-desktop plasma-pa plasma-nm kshutdown dolphin flameshot konsole
```

![plasma](https://i.ibb.co/GJvjCvK/2022-02-24-07-30.png)

Sebenarnya, `plasma-desktop` itu udah cukup, tapi saya tambahin juga program lain agar lebih fungsional

Keterangan:

- `plasma-pa` buat management pulseaudio.
- `plasma-nm` buat network.
- `kshutdown` buat management shutdown, dimana kita bisa mengatur lebih spesifik waktu kapan kita mau shutdown.
- `dolphin` sebagai file manager. Kalo udah punya file manager selain dolphin, silahkan skip.
- `flameshot` untuk ngambil screenshot.
- `konsole` buat terminal. Tapi kalo udah punya terminal lain seperti alacritty, silahkan skip.

Kalo udah terinstall semua, silahkan restart.
